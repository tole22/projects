//////////////////////////////////////////////////////////////////////////////////////
HOW TO INSTALL ANGULAR 4
//////////////////////////////////////////////////////////////////////////////////////

NODE version : for example v 8.11.1 works fine

> npm remove -g @angular/cli                  //remove if you have installed other cli version
> npm install -g @angular/cli@1.1.4           // for angular 4 
> ng --version
  @angular/cli: 1.1.4
> ng new myangular4
